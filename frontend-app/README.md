# 实验过程理解与资产沉淀平台 - 前端

React + TypeScript + Vite 前端应用。

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
npm run preview
```

## 页面

- `/experiments` - 实验列表
- `/experiments/:id/workspace` - 实验工作台
- `/experiments/:id/timeline` - 独立时间线
- `/experiments/:id/steps/:stepId` - 步骤详情
- `/experiments/:id/json` - JSON 查看器
- `/upload` - 新建实验
