# 角色七：数据标注员（Data Annotator）

## 职责定位

负责实验室视频帧的目标检测标注，保证标注质量与类别一致性，持续扩充训练数据集。

## 核心工作内容

### 标注平台
- 主平台：Roboflow（云端管理）
- 导出格式：YOLOv8（YOLO txt 格式）
- 导出后放置路径：`data/dataset/`（运行 `/dataset-add` 合并）

### 13 类标注规范

| 类别 ID | 类别名 | 标注要点 |
|--------|--------|---------|
| 0 | balance | 含完整风罩的天平整体框，不单独框秤盘 |
| 1 | beaker | 烧杯含杯口至杯底，半遮挡时框可见部分 |
| 2 | gloved_hand | 只框戴手套的手，裸手不标注 |
| 3 | lab_coat | 框白大褂的可见躯干部分，不含头部 |
| 4 | paper | 称量纸/滤纸/标签纸均标注，面积 > 50px×50px |
| 5 | reagent_bottle | 棕色/深色试剂瓶，含瓶盖 |
| 6 | sample_bottle | 透明/橙色样品瓶，含瓶盖 |
| 7 | sample_bottle_blue | 蓝盖样品瓶，与 sample_bottle 区分靠盖色 |
| 8 | spatula | 药匙，含勺头至柄末端 |
| 9 | tube | 离心管/试管，含管盖 |
| 10 | tube-cap | 单独的管盖（脱离管体时） |
| 11 | spearhead | 枪头（移液器枪头），细长锥形 |
| 12 | pipette | 移液器整体，含枪头部分 |

### 质量标准
- bbox 与目标边界的 IoU ≥ 0.85
- 遮挡 < 50% 的目标必须标注
- 遮挡 ≥ 50% 的目标可选标注（标注时加 `occluded` 属性）
- 同一目标在视频连续帧中标注框应平滑过渡

### 数据优先级（当前缺口）
优先补充：
1. `tube`（class 9）：目前 0 条
2. `tube-cap`（class 10）：目前 0 条
3. `spearhead`（class 11）：目前 0 条
4. `pipette`（class 12）：目前 0 条

每类目标：train ≥ 300 条，val ≥ 50 条。

### 标注→数据集流程
1. Roboflow 完成标注
2. 导出 YOLOv8 格式 zip
3. 通知检测工程师运行 `/dataset-add <zip路径>`
4. 检测工程师验证后触发重训练

## 每周工作量参考
- 熟练标注员：200-400 帧/天
- 新类别（tube/pipette）：重点采集移液与稀释操作视频帧
