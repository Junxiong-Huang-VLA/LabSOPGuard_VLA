LabSOPGuard

实验室 SOP 合规智能监控系统

项目介绍

面向实验室场景的实时监控平台，支持视频采集、多层级检测、违规识别、告警归档与自动报告生成。

模块结构

  • Video: 视频流采集与预处理
  • Detection: 轻量模型 + 动作分析 + VLM 接口
  • Monitoring: SOP 规则引擎与异常判定
  • Alerting: 实时告警与事件归档
  • Reporting: 报告输入构建与 PDF 生成

环境配置步骤

  1. 运行 `scripts/setup_env.ps1` 或 `setup_env.sh`
  2. 复用或创建 `LabSOPGuard` 环境
  3. 运行 `14_check_environment.py` 完成依赖检查

数据规范

视频、SOP 文档、结构化标注统一放在 `data/raw`，输出事件统一为可追溯 JSON/CSV。

检测与告警流程

Capture -> Detection -> SOP Engine -> Violation -> Alert -> Archive

报告生成流程

Events -> Report Input Builder -> PDF/TXT -> outputs/reports

GitHub 使用建议

  • 主分支保护 + PR 审核
  • 规则变更与文档同步提交
  • 实验记录模板持续维护
